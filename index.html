What This Is

I post thoughts on occasion about whatever's on my mind, and my mind wanders a lot.

All the HTML and CSS are written by your's truly, and can be found here. The styling used to be a great deal more complicated (270 lines of CSS), but I realized that it was a lot of overkill, and so have greatly simplified things. So no more JS, no more Google Fonts, and a lot fewer colors and distracting styles.

The font size may grate on your aesthetic sensibilities, but I find that smaller type can be read with greater speed both in print and on screen, so I prefer it for that reason.
